# How to Create a CLASSIC GitHub Token

## Step-by-Step:

1. **Go to GitHub Tokens Page**
   - Visit: https://github.com/settings/tokens

2. **IMPORTANT - Select "Generate new token (classic)"**
   - You should see two options at the top:
     - ✅ "Generate new token (classic)"  <-- SELECT THIS ONE
     - "Generate new token (fine-grained)"
   - Click on "Generate new token (classic)"

3. **Configure Token**
   - **Note**: Enter "MedDrive Push"
   - **Expiration**: Select "No expiration"
   
4. **Select Scope**
   - Find "repo" in the list and check it ✅
   - This gives full control of private repositories

5. **Repository Access**
   - Select **"All repositories"**

6. **Generate**
   - Click green "Generate token" button

7. **Copy & Provide**
   - Copy the token (starts with `ghp_`)
   - Come back and provide it here
