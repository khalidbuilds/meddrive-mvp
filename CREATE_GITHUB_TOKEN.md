# How to Create a GitHub Personal Access Token

## Step-by-Step:

1. **Go to GitHub Settings**
   - Visit: https://github.com/settings/tokens
   - Or click your profile picture → Settings → Developer settings → Personal access tokens

2. **Generate New Token**
   - Click "Generate new token (classic)"

3. **Configure Token**
   - **Note**: Enter "MedDrive Push" (or any name you prefer)
   - **Expiration**: Select "No expiration" (or 30 days)
   - **Select scopes**: Check ✅ "repo" (Full control of private repositories)

4. **IMPORTANT - Repository Access**
   - Select **"All repositories"** (this is required to push to meddrive-mvp)

5. **Generate**
   - Click "Generate token" button at the bottom

6. **Copy the Token**
   - **IMPORTANT**: Copy the token immediately - it will only be shown once!
   - It will look like: `ghp_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`

7. **Provide the Token**
   - Come back here and provide the token

## Quick Link:
👉 https://github.com/settings/tokens/new?scopes=repo&description=MedDrive+Push
